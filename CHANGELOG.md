# 変更履歴

このページでは、利用者に影響する主な変更をバージョンごとに記録します。
バージョン番号は [Semantic Versioning](https://semver.org/lang/ja/) に準拠します。

## [Unreleased]

### Highlights

- 完成済みの16:9スライド全33枚を、スマートフォンで最大400%まで拡大・移動できるReader Viewを追加
- 章扉、高密度スライド、タブ、出典表示の構成とアクセシビリティを改善
- 「足場・ループ」章の末尾に、2026年7月に急浮上した新語「Graph Engineering」の動向と、暫定的な図解を扱う2枚を追加

### Added

- Reader Viewに全33枚のサムネイル目次、全文検索、補足・出典、対応ブラウザでの全画面表示、キーボード操作を追加
- 旧縦向けReaderと検証用pilotを比較用URLとして保持
- 「『Loopの次はGraph』? まだ数日の“暫定”ワード」「『Graph』は複数のLoopをつなぐ配線」の2枚を追加し、一次情報（本人の投稿・LangGraph公式ドキュメント）を出典として明記
- 参考・出典に「グラフの兆し」グループ、用語集に「グラフエンジニアリング（未確定）」を追加（用語集は18→19語）

### Changed

- 全5章の章扉を、装飾・タイトル・案内を含む中央軸の構成へ統一
- REST APIとMCPの比較を2状態の切替表示へ分け、要求・処理・結果の詳細なアニメーションを片方ずつ追える構成へ変更
- MCP、単一／マルチエージェント、4つの構成パターンに、主体・境界・処理方向を比較できる図解を追加
- スマホReaderをiframe方式から通常Slidevへの直接統合へ変更し、閲覧面が入れ子に見える構造を解消
- GitHub Pagesのbase path、deep reload、hash/history navigationへの対応を強化
- 目次・章内ルート・参考出典・用語集の枚数表記をスライド追加後の全33枚に合わせて更新し、Reader・比較用pilotの対応スライド番号を再採番

### Fixed

- 表紙でGraph Engineeringを確立済み4段階の「5番目」ではなく、既存Loop群を包む暫定topologyとして明示し、全5章扉のタイトル・リード・章内routeが重ならない共通gapと上下余白へ修正
- タブ切替時にタイトルや外枠がわずかに動く問題と、MCP図の枠外へ文字がはみ出す問題を修正
- 表紙に残っていた旧「スマホで拡大」ボタンを削除し、Readerへの公開導線をREADMEへ整理
- Readerの全画面操作を機能対応時だけ表示し、判別しにくい記号を「全画面」／「終了」ラベルへ変更
- Reader目次を折返しの多い2列カードから、サムネイルとタイトルを横に並べた1列リストへ変更
- 拡大リセットの「全体」を「全体表示」＋現在倍率へ分け、操作の意味を明確化
- 旧縦向けReaderのスライド見出しに`word-break: auto-phrase`が未適用だったため、新しいスライド見出しの「まだ」が「ま」「だ」に分断されていたのを修正（全33ページの見出しを再検証・回帰なしを確認）
- 用語集を19語に対応させた際に生じた定義文の尻切れ（2〜3文字だけの行）を、意味を変えない範囲の言い回し短縮で解消し、英字複合語「Zero-shot」の語中改行を改行不可ハイフンで防止
- 上記の修正後に独立した第三者検証（実レンダリングのRange走査）を実施し、同根の語中分断3件を追加で発見・修正: 本編スライドの帰結行（「複数」→「複」／「数の」）、旧Reader用語集オーブ図のバッジ見出し（「エンジニアリング」の分断）、旧Readerのcover/closing帯（「コンテンツ」→「コ」／「ンテンツ」）。`word-break: auto-phrase`を`.tk`（本編）と`figure[data-reader-visual]`配下全体（旧Reader）へ拡張し、外来語複合語は`<wbr>`で意味の継ぎ目を明示。再発防止のQAも追加
